BRAVELY DEFAULT

Bravely Default is a Japanese RPG built by Square-Enix for the Nintendo 3DS. The site features rich illustrations and graphics keeping true to the Final Fantasy series. Nintendo has very strict static requirements. The Bravely Default website was written in static HTML, CSS and JQuery, and leverages both GA and Opengraph Tracking.
